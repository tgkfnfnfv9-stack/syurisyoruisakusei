# 小林機械 現場ツール

株式会社小林機械の従業員が、現場・事務で使う業務ツール集です。
すべて**単一HTMLファイル**・**スマホ（iPhone）対応**で、作成した書類はその場でPDF保存できます。

🔗 公開ページ: <https://tgkfnfnfv9-stack.github.io/>
🏢 会社サイト: <https://www.kkmt.co.jp>

---

## 事業内容

中古の工作機械全般の**買取・修理・オーバーホール**、**改造してからの再販売**を行っています。
外注修理も承ります。また、KBK系列会社に**重量屋（重量物運搬）**があります。

## ツール一覧

| ツール | ファイル | 内容 |
|---|---|---|
| 見積書 | `見積書.html` | 拠点・距離・人員から見積書を作成し、PDF保存 |

> リポジトリ直下に置いてあるファイルに合わせて、この表と `index.html` のリンクを調整してください。

## 使い方・注意点

- **iPhone の Safari 推奨**です。
- 写真添付・在庫検索など**社内サイト（kkmt.co.jp）を開く機能**は、**ログイン済みの Safari** で使ってください。Documents by Readdle などのアプリ内ブラウザでは、ログイン情報が無く開けないことがあります。
- 見積書の「距離を自動取得」を使う場合は、Google Cloud で **Routes API を有効化**し、**APIキー**を各ツールの入力欄に貼り付けてください（キーは端末内にのみ保存されます）。
- 各ツールは**ネット接続なしでも基本動作**します（バーコード読み取りも含む）。

---

## 技術メモ

- 単一HTMLファイル・**CDN非依存**（外部読み込みなし＝オフライン動作）
- PDF生成: jsPDF + html2canvas
- バーコード読み取り: zxing-wasm（C++ WASM）
- データはブラウザ内（localStorage）にのみ保存。サーバー送信なし。
